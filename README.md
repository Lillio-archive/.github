# Lillio Archive

This org is the retirement village for repositories that put in the work, shipped the features, survived the incidents, and earned a well-deserved rest.

Every repository listed here is archived, with one small exception: `.github`, the special repository that keeps this profile README visible. Some repos are public, some are private, and private links require org access.
